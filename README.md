## How to Run
Clone
```sh
$ git clone https://github.com/Ranatchai/co-study-prototype
```
Install Dependencies
```sh
$ cd co-study-prototype && npm install
```
Run
```sh
$ ./bin/www
```

Then, visit http://localhost:9091 to view a result.
